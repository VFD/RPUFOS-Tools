Introduction

Ensemble d'utilitaire en Javascript pour développer des applications sur les Obscures Systèmes.

L’un des grands avantages de JavaScript est qu’il peut s’exécuter pratiquement partout aujourd’hui.

JavaScript offre aussi une excellente maintenabilité.

Vous pouvez soit ouvrir directement le fichier index.html, soit démarrer un serveur web local, soit déployer l’ensemble sur un site web.

Cliquer sur l'icone RPUFOS en haut à gauche pour revenir sur cette page.

Methode canadienne, l'intro à du Français, tout le reste en US.

Introduction

A set of JavaScript utilities for developing applications on the Obscure Systems.

One of the major advantages of JavaScript is that it can run virtually anywhere nowadays.

JavaScript also offers excellent maintainability.

You can either open the index.html file directly, start a local web server, or deploy the whole project to a website.

Click the icon RPUFOS in the upper left corner to return to this page.

Installation

L’installation est simple : téléchargez le dépôt Git, décompressez-le, puis ouvrez index.html dans votre navigateur.

Pour exécuter en local, des scripts sont fournis en fonction du système d’exploitation et des outils disponibles (PowerShell, Python, etc.).

Installation

Installation is straightforward: download the Git repository, unzip it, and open index.html in your browser.

For local execution, scripts are provided depending on the operating system and the available tools (PowerShell, Python, etc.).
