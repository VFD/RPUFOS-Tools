Online Assemblers by Architecture

Here is a curated list of online assemblers and IDEs for different retro architectures.
These links provide direct browser-based tools to complement your suite without extra maintenance.
This needs to be improved.

Multi‑CPU

  • ASM80: asm80.com — Web IDE supporting Z80, 6502, 6800/6809 with assembler, editor, and export.
  • EightBitWorkshop:eightbitworkshop.com — Browser IDE for 8‑bit consoles/computers (NES/6502, Atari/6502, etc.) with emulator integration.

Z80

  • ASM80 (Z80): asm80.com — Z80 assembler with macros and export; useful for CPC, Spectrum, MSX.
  • EightBitWorkshop (Z80):eightbitworkshop.com — Z80 projects depending on supported targets and templates.

6502 / 65C02

  • Masswerk 6502 Assembler: masswerk.at/6502/assembler.html — Online assembler and simulator, great for quick tests.
  • EightBitWorkshop (6502): eightbitworkshop.com — Full environment with compilation and execution on 6502 targets.
  • ASM80 (6502): asm80.com — 6502/65C02 support with binary export.

6809

  • ASM80 (6809): asm80.com — 6809 support, useful for Tandy/Dragon and Thomson machines.

TMS7000 / TMS7020 / TMS7041

  • ASM7000: github.com/GmEsoft/ASM7000 — Lightweight assembler for the TMS7000 family, generates binaries for EXL100 and related machines.
  • libasm: github.com/tgtakaoka/libasm — Multi‑CPU assembler library including support for TMS7000 series.

x86 (NASM/ASM)

  • Compiler Explorer (NASM): godbolt.org — NASM support with machine code visualization and disassembly.
  • PCjs Machines: pcjs.org — IBM PC environments in the browser; useful for testing binaries and DOS tools.
